Projeto simples de análise de dados utilizando Python.

O programa trabalha com uma lista de valores simulando dados de vendas e realiza cálculos básicos para análise dessas informações.

Entre as análises realizadas estão:
- Cálculo do total de vendas
- Quantidade de registros
- Média dos valores
- Identificação da maior venda
- Identificação da menor venda

O objetivo do projeto é praticar manipulação de dados e funções básicas da linguagem Python aplicadas à análise de dados.
